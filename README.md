---------------------TheServiceGETip---------------------------------------
TheServiceGETip.py 可以获取外网IP并发送到邮箱。

安装服务：python TheServiceGETip.py install
启动服务：python TheServiceGETip.py start
停止服务：python TheServiceGETip.py stop
删除服务：python TheServiceGETip.py remove
